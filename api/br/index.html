br

br is the global table which implements the BadRotations API. Due to its unavoidable size, complexity, and mutability, its members are varied and logically defined within many files.

In this documentation, some members may be left undocumented for brevity. This is not to say that they are not important, but rather that they are internal or not intended for public use. If you find yourself using an undocumented member, please consider opening an issue on GitHub to request that it be documented.
